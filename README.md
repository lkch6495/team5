
# team5 프로젝트

·구성원 및 역할 분담: 이서진(기획), 원준영(디자인), 김태강(자료수집)

·소개: 이 웹페이지는 1세대부터 5세대까지의 K-POP의 흐름을 다루며, 각 세대의 특징을 소개한다. 또한 세대를 대표하는 그룹들의 자세한 정보, 특징, 대표곡에 대한 설명을 함께 제공하여 K-POP에 대한 흥미와 이해를 높이고자 한다.

·사용방법: zip 압축파일로 다운을 받고난 후, 압축을 해제한다. 그 후 indexoo.html파일을 열어 기본 내용과 그 밑에 있는 각 카드를 클릭하여 다른 웹페이지로 접속하여 내용을 확인할 수 있다.

코드 기본 구조는 Chat GPT를 이용하여 작성함.
